Stored information

We keep limited information about the user stored, all for the purposes of supporting application features:

  • Facebook User ID - For the purposes of tracking a user's high score, and in-app purchases
  • Facebook User Name and IP - Appear in server logs for debugging and trackback purposes
  • Comments / Suggestions / Bugs - For the purposes of improving the user experience
  • OS and Browser Information - For debugging purposes, included with bug reports

Removal

To ensure the removal of your personal information from our server, uninstall the application from the Facebook application settings.

Changes to Policy

This policy is updated as we need to collect new or different information to implement new features.

Disclosure

It is our policy to not disclose your personal information to any third party.
